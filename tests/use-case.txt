Feature: 사용자 로그인
  Scenario: 유효한 정보로 로그인 성공
    Given 사용자가 로그인 페이지에 접속한다
    When 사용자가 아이디 입력란에 "valid_user"를 입력한다
    And 사용자가 비밀번호 입력란에 "valid_password"를 입력한다
    And 사용자가 "로그인" 버튼을 클릭한다
    Then 사용자는 "대시보드" 페이지로 이동한다
    And 사용자는 "환영합니다, valid_user님!" 메시지를 본다

Feature: 채팅 기능
  Scenario: 특정 채팅방에 메시지 전송 및 확인
    Given 사용자가 로그인하여 채팅 목록 화면을 보고 있다
    And 채팅 목록에 "스터디 그룹" 채팅방이 표시된다
    When 사용자가 채팅 목록에서 "스터디 그룹" 채팅방을 클릭하여 진입한다
    And 사용자가 메시지 입력 필드에 "안녕하세요, BDD 테스트 중입니다!"라고 입력한다
    And 사용자가 "전송" 버튼을 클릭한다
    Then "스터디 그룹" 채팅방의 대화 내용 영역에 "안녕하세요, BDD 테스트 중입니다!" 메시지가 마지막 메시지로 표시된다
